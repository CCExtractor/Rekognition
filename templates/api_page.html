POST /api/image

currently It returns the face recognized and bounding box

example
input 
curl -F 'file=@/Users/amit/Desktop/11.jpg' http://127.0.0.1:8888/api/image
output 

							{
								"identity" : "Elon Musk",
								"Bounding Box" :
									{
										"top" : 61,
										"bottom" : 313,
										"left" : 88,
										"right" : 291
									}
							}
POST /api/video

currentlry It returns the name of the recognized faces and at what interval in the video they did appear.

example
input 
curl -F 'file=@/Users/amit/Desktop/test.mp4' http://127.0.0.1:8888/api/video
output 

						{
							"face_id_1":[[0.3,0.3],[0.8,8.51],[22.22,25.53],[28.63,31.43]]
							"face_id_1":[[0.1,0.7],[1.2,1.4],[8.61,10.41],[11.31,12.51],[12.81,22.12],[25.63,28.53],[31.53,35.34],[35.84,36.24]]
						}

Here all the values are in sec , "face_id_1":[0.8,8.51]", It means face appeared from 0.8 >----> 8.51 secs.
